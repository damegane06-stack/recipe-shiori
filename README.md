# レシピの栞 〜料理リンクまとめ帖〜

X（Twitter）や Instagram、YouTube、TikTok などで見つけた料理を保存して、
「あとで作る」を忘れないための、個人用レシピ管理アプリです。

黒板＋和紙をイメージした和モダンなデザインで、スマホでも片手で使えるように最適化しています。

## 特長

- **インストール不要** — `index.html` 1ファイルだけで動作。ビルドツールもサーバーも不要です。
- **データはブラウザに保存** — 保存先は `localStorage`。アカウント登録なしで使えます。
- **スマホ最適化** — 画面下に固定されたナビで、親指だけで操作できます。

## 使い方

`index.html` をブラウザで開くだけで使えます。
スマホのホーム画面に追加すると、アプリのように起動できます。

画面下部の 4 つのタブで機能を切り替えます。

### 📋 一覧

保存したレシピをサムネイル付きのカードで一覧表示します。

- ジャンル別タブ・小ジャンルでの絞り込み
- タイトル / タグでの検索
- 難易度・作りたい度（🔥）でのフィルタ
- お気に入り（★）だけを表示

カードをタップすると詳細（材料・メモ・調理記録など）が開きます。

### ➕ 追加

レシピを新しく登録します。

| 項目 | 内容 |
|------|------|
| リンク | X / Instagram / YouTube / TikTok などの URL。プラットフォームを自動判定します |
| タイトル・料理名 | 料理の名前 |
| ジャンル / 小ジャンル | 肉・魚介・麺・ご飯もの など（自由に管理・追加できます） |
| 調理時間・難易度 | 5分以内〜1時間以上 / 初心者OK・普通・上級 |
| 作りたい度 | 🔥 1〜3 段階 |
| 材料メモ | 改行で 1 行 1 品。**「材料」と「調味料」を自動で見分けます** |
| 調理メモ | ポイントや下ごしらえのコツなど |
| フリータグ | 「弁当OK」「夏向け」などスペース区切りで複数登録 |
| サムネイル | YouTube・TikTok はリンクから自動取得。Instagram・X はスクショ添付がおすすめ |

### 🛒 買い物

買い物リストを管理します。

- 食材を手入力で追加
- 保存したレシピの「材料」と「調味料」をまとめてリストに取り込み
- チェック済みのまとめ削除・全削除

### 🎲 献立

「今日は何を作る？」を、条件（ジャンル・調理時間・難易度）で絞り込んで
ランダムに 1 品提案します。

## 便利な機能

- **カロリー自動計算** — 材料メモを解析し、食品成分データベース準拠の kcal 辞書と単位変換（大さじ→15g など）から、レシピ全体のカロリーを推定表示します。
- **材料と調味料の自動仕分け** — 材料メモの各行を判定し、色分け表示。買い物リストへの取り込みにも使われます。
- **作った記録** — 作った日付・5 段階評価・ひとことメモを履歴として残せます。
- **クラウド同期** — GitHub Gist を使って、スマホと PC でデータを共有できます（後述）。
- **書出 / 読込** — データを JSON でバックアップ・復元できます。

## クラウド同期（任意）

複数の端末でデータを共有したい場合は、GitHub Gist 経由で同期できます。

1. 一覧画面の「☁️ 同期」を開きます。
2. GitHub の **Personal Access Token（`ghp_〜`。gist 権限）** と **Gist ID** を入力します。
3. 「設定を保存して同期する」を押すと、以降は自動で同期されます。

> トークンはブラウザの `localStorage` に保存されます。共用端末では利用にご注意ください。

## データについて

- レシピ・ジャンル・買い物リストは、それぞれ `localStorage` の
  `rs_recipes` / `rs_genres` / `rs_shop` キーに保存されます。
- ブラウザのデータを消去すると失われるため、大切なデータは「書出」で
  定期的にバックアップすることをおすすめします。

## 技術構成

- HTML / CSS / バニラ JavaScript のみ（フレームワーク・依存ライブラリなし）
- 外部読み込みは Google Fonts（Kaisei Decol / Zen Kaku Gothic New）のみ
- 保存: `localStorage` ＋ 任意で GitHub Gist 同期
